# Student Management System

## Overview

A backend Student Management System built using Spring Boot and MySQL. The application provides REST APIs for performing CRUD operations such as adding, retrieving, updating, and deleting student records.

## Technologies Used

* Java
* Spring Boot
* MySQL
* Spring Data JPA
* Hibernate
* Maven
* Postman

## Features

* Add Student
* View Students
* Update Student
* Delete Student

## API Endpoints

### Get All Students

GET `/students`

### Add Student

POST `/students`

### Update Student

PUT `/students/{id}`

### Delete Student

DELETE `/students/{id}`

## Testing

APIs were tested using Postman.

## Database

MySQL database integration using JPA/Hibernate ORM.
